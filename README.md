# uperf-12x

Uperf-12X custom build for Snapdragon 8 Elite Gen5 (8EG5 / SM8850), focused on battery-efficient smoothness.

## Included in this repo

- 8EG5 scheduler config and per-app policy rules
- Closed-loop CPU scheduler script (frame-target + tail-latency/energy joint score)
- Module scripts and metadata for Magisk/KernelSU packaging

## Notes

- This repository currently tracks the editable module sources and scripts.
- Prebuilt binaries (`bin/uperf`, `bin/busybox/busybox`) are not uploaded through this connector flow.
